# Industrial Design Portfolio Homepage

This is a modern, responsive homepage for an industrial design portfolio. It features a hero section, featured projects, about, and contact info.

## How to Run

1. Place your project images as `project1.jpg`, `project2.jpg`, and `project3.jpg` in the same folder.
2. Open `index.html` in your browser.

## Customization
- Update your name, bio, and contact info in `index.html`.
- Replace project images and descriptions as needed.
- Style changes can be made in `style.css`.
